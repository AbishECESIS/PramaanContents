Pramaan: Your ONDC integration Test Bench

Test, Fix, and Accelerate your integration with ONDC network

ONDC integration test bench is designed to ensure seamless interoperability and complaince with the ONDC protocols. It enables developers to test and validate their ONDC-based applications, services, and integration through defined flows & scenarios.

Why to use Pramaan: ONDC Integration Test Bench?

  • Baseline Test Flows: Pramaan verifies baseline scenarios and flows required to enable operations on ONDC network.

  • Simulation of ONDC Network: Emulates the ONDC network, allowing testing of complex scenarios and businesses cases across APIs.

  • Multi-Protocol Support: Tests integration across live ONDC protocols (e.g., Retail, mobility, logistics, financial services for buyer & seller apps).

  • Real-time Feedback: Provides instant test results, identifying errors and areas for improvement.

  • Regression test suite: You can perform integration check through this regression test suite anytime after go-live.

  • Reduce integration errors: Review Pramaan Integration Report to confirm implementation of baseline flows by other NPs.

How Pramaan Works?

Step 1

Enable prerequisites are done based on domain

Review prerequisites based on domain. Go through the demo videos to understand how to run and use Pramaan before you begin testing.

Step 2

Provide details about your Application

Provide subscriber ID as per chosen environment - Staging or Preprod. Details about application & testing configuration to setup the test runs.

Step 3

Run Tests

Follow guided steps to execute tests. There are steps which require actions to be carried out on your application in a time-bound manner.

Step 4

Fix & Repeat

Download and review the report after run and resolve any issues, if found. Re-run after fixing the issues till an all-clear report is generated.

Successful Pramaan runs get synched with ONDC Participant Portal. You can download consolidated reports from the portal.
